# Configuración de Proxies

Esta carpeta contiene la configuración de proxy inverso para tres servidores web diferentes:

- **Caddy**: Configuración moderna y automática con soporte integrado para HTTPS.
- **Nginx**: Archivos de configuración para proxy inverso de alto rendimiento.
- **Apache**: Configuración mediante módulos como `mod_proxy` para redirección de tráfico.

Cada subdirectorio incluye los archivos necesarios y ejemplos para desplegar el proxy correspondiente.
